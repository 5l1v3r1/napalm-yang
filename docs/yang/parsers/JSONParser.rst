JSONParser
==========

This extractor will read an XML an extract data from it.

To illustrate the examples below we will use the following configuration::

    <configuration>
        <interfaces>
            <interface>
                <name>ge-0/0/0</name>
                <description>adasdasd</description>
            </interface>
            <interface>
                <name>lo0</name>
                <disable/>
            </interface>
        </interfaces>
    </configuration>
